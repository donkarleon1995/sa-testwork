## 2. Исправленная версия требований

1. Пользователь может добавить в корзину от 1 до 10 единиц одного товара.
2. Пользователь может изменять количество товара в корзине в диапазоне от 1 до 10.
3. Для удаления товара из корзины используется кнопка «Удалить».
4. В корзине может находиться не более 5 различных товаров.
5. Суммарное количество всех товаров в корзине не может превышать 20 единиц.
6. При попытке добавить товар или изменить его количество с превышением любого из лимитов (п.1, п.4, п.5) система отображает сообщение:
«Лимит корзины превышен».
7. Цена товара фиксируется на момент добавления в корзину и не изменяется при изменении цены в каталоге.
8. На странице корзины отображаются:
   - список товаров,
   - количество каждого товара,
   - цена за единицу,
   - общая стоимость позиции,
   - общая стоимость корзины.
9. В корзине может отображаться рекламный блок с другими товарами.
10. Рекламный блок отображается в будние дни (понедельник–пятница):
   - утром с 08:00 до 11:59,
   - вечером с 18:00 до 21:59
(по локальному времени пользователя).
11. Вне указанного времени рекламный блок в корзине не отображается.

## 3. Уточняющие вопросы заказчику

1. По ограничениям корзины
   - Что важнее при конфликте лимитов: суммарное количество или количество уникальных товаров?
   - Можно ли частично добавить товар (например, 2 из 5), если превышается лимит?
2. По удалению товаров
   - Должен ли пользователь иметь возможность установить количество = 0, или удаление только через кнопку?
   - Нужно ли уведомление о подтверждение при удалении товара?
3. По цене
   - Цена должна фиксироваться до оформления заказа или до оплаты?
   - Возможна ли ситуация, когда цена обновляется при переходе к оформлению заказа?
4. По рекламе
   - Что показывается в корзине в выходные?
   - Должна ли реклама зависеть от содержимого корзины?
   - Что делать, если пользователь держит страницу открытой и время показа рекламы изменилось?
5. Общее
   - Корзина привязана к пользователю или к сессии?
   - Нужно ли сохранять корзину между устройствами?
